# Hello World Flask Application
## Checkout this repo and switch to branch
```shell
git clone https://github.com/windson/flask-tutorials.git
cd flask-tutorials
git checkout hello-world-flask
```
## Create Virtual Environment
Windows OS

```shell
python -m venv env
```

Unix OS Variants

```shell
python3 -m venv env
```

## Activate Virtual Environment

Windows OS

```shell
env\Scripts\activate
```

Unix OS Variants

```shell
source ./env/bin/activate
```

## Upgrade pip

```shell
python -m pip install --upgrade pip
```

## Install Requirements for requirements.txt

```shell
pip install -r requirements.txt
```

## Run Flask App :rocket:

```shell
python app.py
```

## Access the App in the browser
Access the url `http://localhost:5000/` in the favorite browser üòç of your choice.
